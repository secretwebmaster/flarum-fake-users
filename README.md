# Fake user generator

A [Flarum](http://flarum.org) extension. 


### Installation

Install manually with composer:

```sh
composer require secretwebmaster/flarum-fake-user:"*"
```

### Updating

```sh
composer update secretwebmaster/flarum-fake-user
```
