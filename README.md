# Fake user generator

A [Flarum](http://flarum.org) extension. 


### Installation

Install manually with composer:

```sh
composer require secretwebmaster/flarum-fake-users:"*"
```

### Updating

```sh
composer update secretwebmaster/flarum-fake-users
```
